## Core Module

The **Core Module** serves as the backbone of the application, providing essential functionality and integration with other modules.

### Features

1. **Module Loading**:
   - Dynamically load and manage other application modules.
   - Ensure seamless integration and communication between modules.

2. **Cron Job Scheduling**:
   - Support for scheduling tasks using **Celery**.
   - Enable automation of periodic tasks such as data scraping and processing.

3. **Reusable Functionality**:
   - Provide common, reusable utilities and services that can be leveraged by other modules.
